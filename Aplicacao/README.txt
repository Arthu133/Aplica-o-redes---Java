Passos para rodar o chat:

1- Abrir o ServidorChat na máquina que vai ser o host do chat em texto
2- Trocar a variável ENDERECO_SERVIDOR_TEXTO para o IP do servidor de texto
3- Na máquina 1, trocar a variável ENDERECO_SERVIDOR_AUDIO para o IP da máquina 2
4- Abrir o TudoChat na máquina 1
5- Na máquina 2, trocar a variável ENDERECO_SERVIDOR_AUDIO para o IP da máquina 1
6- Abrir o TudoChat na máquina 2
7- Escrever uma mensagem na caixa
8- Apertar o botão ">" (enviar mensagem TCP)
9- Conversar via áudio (UDP)
